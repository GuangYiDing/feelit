# Feel it

> 当你无法控制自己的身体时,是什么感觉?

感慨易烊千玺在 [小小的我](https://m.douban.com/movie/subject/36498717/) 中的精湛演技
开始思考这个问题,于是有了这个工具

点击开始按钮,30s 内鼠标将会随机抖动,试着让它停下来

![demo](./doc/demo.gif)

### 为什么叫 feel it

`it`指代意识清醒,身体却无法完美控制的感觉

**理解和尊重**的最好办法是`感同身受`
